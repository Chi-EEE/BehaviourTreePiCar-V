This sub-\/project handles all the logic of the Sunfounder Raspberry Pi.

The system is broken down to 3 sub systems\+:
\begin{DoxyItemize}
\item lidar -\/ Handles the lidar scanner of the Raspberry Pi
\item messaging -\/ Handles the websocket of the Raspberry Pi
\item movement -\/ Handles the movement of the Sunfounder Raspberry Pi
\end{DoxyItemize}

The system is then displayed by the {\ttfamily Car\+Console}.\hypertarget{md_tui_README_autotoc_md17}{}\doxysection{Installation}\label{md_tui_README_autotoc_md17}
Run the following command to build the sub-\/project\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{xmake build raspberry\_pi}

\end{DoxyCode}


To connect the Raspberry Pi to the backend, you would need the {\ttfamily IPv4 Address} of your Computer/\+Server and change the host to the value in {\ttfamily IPv4 Address}.
