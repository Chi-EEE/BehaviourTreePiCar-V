\hypertarget{md_daemon_install_README_autotoc_md1}{}\doxysection{Installation\+:}\label{md_daemon_install_README_autotoc_md1}
Run the follow commands to install the Daemon\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{sudo chmod +x ./install\_script.sh \# Grant permission to install\_script to run as exe}
\DoxyCodeLine{systemctl stop rpi\_daemon \# Stop any previous rpi\_daemon}
\DoxyCodeLine{sudo ./install\_script.sh install \# Install the Daemon}
\DoxyCodeLine{sudo systemctl daemon-\/reload \# Reload the Daemon if there was any previous}
\DoxyCodeLine{sudo systemctl start rpi\_daemon \# Start the new Daemon}

\end{DoxyCode}


Or use the single command\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{sudo chmod +x ./install\_script.sh \&\& systemctl stop rpi\_daemon \&\& sudo ./install\_script.sh install \&\& sudo systemctl daemon-\/reload \&\& sudo systemctl start rpi\_daemon}

\end{DoxyCode}
